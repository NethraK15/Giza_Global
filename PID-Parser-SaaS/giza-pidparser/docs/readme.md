# P and ID Parser Version 1.0

### Django UI setup:
* Download the Project P&ID parser folder.

* Open the folder in VScode, open cmd prompt.

* Move to the Backend directory.
> cd app/backend

* Run the build.bat file
> build.bat

*Access the Project
![access](assets/fig-01-acess-image.jpg)

* Login to page
**Username:** admin
**Password:** demodemo

* Enter the file name and choose the file to be parsed and click the submit button saves the uploaded file in the system for later use and then click Parse file button which will initiate the processing of the most recent upload.

* If you have already uploaded the files, click the respective detect button in the displayed uploaded files table
