About Models

Dataset Generation

The dataset used for training the models was generated using Roboflow, a powerful tool for annotating images and creating datasets for computer vision tasks.

YOLO Models

Three YOLO (You Only Look Once) models were used for object detection. YOLO is a state-of-the-art, real-time object detection system known for its speed and accuracy. It can detect multiple objects in an image simultaneously, making it highly efficient for complex detection tasks. The accuracy of the three models is approximately as follows:

  • Model 1: 98% for Columns, vessels, instruments, etc.
  • Model 2: 98% for LMIs, Tags, PLCs, etc.
  • Model 3: 86% for valves, reducers.

Model Segmentation

The use of three different models was necessitated by the varying sizes of symbols (e.g., valves) within the images. Each detection model was tailored to a specific segmentation of the image, enhancing detection performance for differently sized objects.

Line Detection

For detecting lines, the images were processed using skeletonization and morphological operations. These techniques help in highlighting and extracting line structures from the images, achieving high detection accuracy. Skeletonization reduces the lines to a single-pixel width, while morphological operations enhance and isolate the lines for accurate detection.

Classification of Valves

A CNN model is used to classify the detected valves with 90% accuracy.

Graph Generation

The graphs are generated to visualize the detection and classification results.
